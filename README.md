# PhoenixC2
> An Open-Source C2-Framework written in pure Python3.

## Features
- [x] Modern Web-Interface
- [x] CLI-Interface
- [x] Built for Teams and Organizations (Multi-User)
- [x] Customizable (Plugins, Modules, Kits)
- [x] Easy to use
- [x] Easy to extend
- [x] Supports different languages (Python, C#, Go, ...)
